HOW TO RUN LOCALLY
---
METHOD 1: Python HTTP Server (Recommended)
1) Open terminal in the project folder
2) Run this command:

   python -m http.server 8000 --bind 0.0.0.0

3) Open in browser:
   - Same device:
     http://localhost:8000

   - Another device on the same WiFi:
     http://<LOCAL_IP>:8000
